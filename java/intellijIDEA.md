### 快捷键

| 快捷键                 | 解释                     |
| ---------------------- | ------------------------ |
| ctrl + alt + B         | 查看子类                 |
| alt + shift + ↑↓       | 上下移动当前行           |
| alt + insert           | generator 快速生成       |
| alt + j                | 选中下一个相同内容       |
| ctrl + shift + alt + j | 选中所有相同内容         |
| ctrl + alt + F7        |                          |
| ctrl + F12             | 查看当前类的结构         |
| ctrl + y               | 删除当前行               |
| ctrl + '+/-'           | 当前方法展开、折叠       |
| ctrl + shift + '+/-'   | 全部展开、折叠           |
| ctrl + w               | 选中光标所在字符串       |
| ctrl + g               | 跳转到指定行             |
| ctrl + shift + R       | 全局搜索                 |
| ctrl + shift + U       | 大小写转换               |
| ctrl + shift + Enter   | 补全语句                 |
| ctrl + alt + B/click   | 查看哪些类实现了当前接口 |

### IDEA 中 Terminal 使用 vim 编辑后退出问题

直接按 esc 会和 idea 的快捷键重合，可以使用 ctrl+c 退出 vim 的编辑模式
