### 快捷键

| 快捷键               | 解释               |
| -------------------- | ------------------ |
| ctrl + alt + B       | 查看子类           |
| alt + shift + ↑↓     | 上下移动当前行     |
| alt + insert         | generator 快速生成 |
| ctrl + alt + F7      |                    |
| ctrl + F12           | 查看当前类的结构   |
| ctrl + y             | 删除当前行         |
| Ctrl + '+/-'         | 当前方法展开、折叠 |
| Ctrl + shift + '+/-' | 全部展开、折叠     |
| Ctrl + w             | 选中光标所在字符串 |
| Ctrl + g             | 跳转到指定行       |
| Ctrl + shift + R     | 全局搜索           |
| Ctrl + shift + U     | 大小写转换         |

### IDEA 中 Terminal 使用 vim 编辑后退出问题

直接按 esc 会和 idea 的快捷键重合，可以使用 ctrl+c 退出 vim 的编辑模式
